This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide.

VBAI Body Height

An observation profile based on US Core Body Height Profile
VBAI Body Weight

An observation profile based on US Core Body Weight Profile.
VBAI Condition

A condition profile based on US Core Condition Encounter Diagnosis.
VBAI Consent

A consent profile based on the base Consent resource
VBAI Encounter

An encounter profile based on US Core Encounter.
VBAI Imaging Media

A imaging media profile based on the base Media resource.
VBAI Imaging Study

An imaging study profile based on the base ImagingStudy resource.
VBAI Patient

A patient profile based on US Core Patient
VBAI Questionnaire

A questionnaire profile based on the base Questionnaire resource
VBAI Questionnaire Response

A questionnaire response profile based on the base QuestionnaireResponse resource
VBAI Vital Signs

A observation profile based on US Core Vital Signs Profile.
VBAI Waveform Device

A device profile based on the base Device resource.
VBAI Waveform Media

A media profile based on the base Media resource. Primarily captures binary audio waveform data.
